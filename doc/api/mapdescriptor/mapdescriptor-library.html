MapDescriptor

MapDescriptor is a Dart package that provides a simple way to encode and decode JSON data. With MapDescriptor, you can easily convert between Dart objects and JSON strings, making it easy to work with JSON data in your Dart applications.

Installation

To use MapDescriptor in your Dart application, add the following to your pubspec.yaml file:

yaml
dependencies:
  mapdescriptor: ^1.0.0

Usage

Encoding

To encode a Dart object to a JSON string, simply create an instance of the MapDescriptor class and call its encode method, passing in the object you want to encode:

dart
import 'package:mapdescriptor/mapdescriptor.dart';

void main() {
  final data = {'name': 'John', 'age': 30};
  final json = MapDescriptor().encode(data);
  print(json); // {"name":"John","age":30}
}

Decoding

To decode a JSON string to a Dart object, create an instance of the MapDescriptor class and call its decode method, passing in the JSON string:

dart
import 'package:mapdescriptor/mapdescriptor.dart';

void main() {
  final json = '{"name":"John","age":30}';
  final data = MapDescriptor().decode(json);
  print(data); // {name: John, age: 30}
}

Custom Encoding and Decoding

MapDescriptor also allows you to customize the encoding and decoding process for your own Dart classes. To do this, you need to create a custom encoder and decoder for your class that implements the mapdescriptorEncoder and mapdescriptorDecoder interfaces, respectively.

Here's an example:

dart
import 'package:mapdescriptor/mapdescriptor.dart';

class Person {
  final String name;
  final int age;

  Person(this.name, this.age);

  factory Person.fromJson(Map<String, dynamic> json) {
    return Person(json['name'], json['age']);
  }

  Map<String, dynamic> toJson() {
    return {'name': name, 'age': age};
  }
}

class PersonEncoder implements mapdescriptorEncoder<Person> {
  @override
  dynamic encode(Person data) {
    return data.toJson();
  }
}

class PersonDecoder implements mapdescriptorDecoder<Person> {
  @override
  Person decode(dynamic data) {
    return Person.fromJson(data);
  }
}

void main() {
  final person = Person('John', 30);
  final encoder = MapDescriptor().registerEncoder(Person, PersonEncoder());
  final decoder = MapDescriptor().registerDecoder(Person, PersonDecoder());
  final json = encoder.encode(person);
  print(json); // {"name":"John","age":30}
  final decodedPerson = decoder.decode(json);
  print(decodedPerson); // Person{name: John, age: 30}
}
